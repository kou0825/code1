壁()

壁()とは
　　メモリ[0] → 0
　　メモリ[0] < 4 なかぎり
　　　　メモリ[1] → 0
　　　　メモリ[1] < 4 なかぎり
　　　　　　メモリ[2] → 0
　　　　　　メモリ[2] < 20 なかぎり
　　　　　　　　メモリ[60000 + メモリ[3] + メモリ[0] + ((メモリ[2] * 5 + メモリ[1]) * 100)] → 999999
　　　　　　　　メモリ[60055 + メモリ[3] + メモリ[0] + ((メモリ[2] * 5 + メモリ[1]) * 100)] → 999999
　　　　　　　　メモリ[2] < 10 なら
　　　　　　　　　　メモリ[69505 + メモリ[0] + (メモリ[1] * 100) + (メモリ[2] * 5)] → 999999
　　　　　　　　メモリ[2] → メモリ[2] + 1
　　　　　　メモリ[1] → メモリ[1] + 1
　　　　メモリ[0] → メモリ[0] + 1
